Pakuranga 5km Fun Run and Walk

The local 5km Fun Run and Walk was in need of a new logo. I started this project by mocking up some logos using character design inspired by the style used in the Olympics sport pictograms. I then moved closer to my final outcome by creating the illustrated silhouette of a local club memeber who was a New Zealand Olympic medalist.
